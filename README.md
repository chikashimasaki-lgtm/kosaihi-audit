# 会議費・交際費 日常チェック統合型 監査スクリーニング

経理の日常チェックの中で、交際費の科目分散・過剰接待・相手方偽装・決裁プロセスの逸脱を自動スクリーニングするブラウザアプリ（サーバー不要・単一HTML）。

## 何をするか
入力した精算データに対し、以下を**機械的に確定判定**し、疑義案件のみ**監査内容確認書**を生成します。

1. **税務・会計（STEP1）** … 税抜経理前提。インボイス未登録店は経過措置（2026年＝控除不可20%）を本体に上乗せした「監査用税抜価額」で1人あたり金額を計算し、1万円基準を判定。社外0名は金額不問で交際費。
   - **費目カテゴリ判定**（店舗/内容・申請科目から推定）… 飲食以外も検知。
     - **ゴルフ**（取引先同伴）＝全額交際費。旅費交通費等での計上は隠れ交際費。事前申請なしと併発で要改善。
     - **手土産・贈答**（取引先向け）＝全額交際費。会議費等での計上は隠れ交際費。
     - **旅費**（社外同伴）＝接待旅費の疑い（交際費の可能性）。社内のみ出張は旅費交通費で適正。
2. **決裁プロセス** … 事前承認日＞支出日（事後決裁）・関連決裁番号なし（事前申請スキップ）・精算決裁の遅延を検知。
3. **不祥事シナリオ（STEP2）** … 高級店 × 自社役員 × 社外一般担当 の役職不整合＝相手方偽装の疑い、高級店×事前申請なし＝隠蔽目的の回避を検知。
4. **格付け** … 🟩適正 / 🟦注意 / 🟨要改善(軽微) / 🟥要改善 に自動分類。
5. **STEP3 定性判定（任意）** … 事後決裁理由の妥当性などはAI（Copilot/Gemini）用プロンプトを同梱（アプリ内からコピー）。

## 使い方
1. アプリを開く → 「📋 サンプルデータ投入」or 自社データを貼り付け（タブ/カンマ区切り）
2. 「🔍 監査実行」
3. サマリー・判定一覧・監査内容確認書（疑義のみ）が出力 → 結果CSVもDL可

### 入力列順
`精算申請ID, 関連決裁番号, 支出日, 事前承認日, 精算決裁日, 金額税込, 人数, 社外人数, インボイス(登録/未登録), 店舗, 申請科目, 社内最上位役職, 社外最上位役職, 事後決裁理由`

サンプル: `sample_data.csv`

## 注意
- 一次スクリーニング用。最終判定・税務処理は専門家・原資料で確認すること。
- 機械判定（税抜計算・日付・1万円基準）は確定値、隠蔽/偽装は「疑義の示唆」。
- 入力データはブラウザ内のみで処理され、外部送信はありません。
